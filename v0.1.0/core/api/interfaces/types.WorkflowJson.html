    Interface WorkflowJson<N>

    Our WorkflowJson interface represents a workflow in n8n-compose. THe main difference to the n8n workflow interface is the different representation of connections. We also make more properties optional to make workflow definitions less verbose.

    interface WorkflowJson<N extends readonly NodeBase[]> {
        active?: boolean;
        connections: Connections<N>;
        createdAt?: Date;
        id?: string;
        name: string;
        nodes: N;
        pinData?: IPinData;
        settings?: IWorkflowSettings;
        startedAt?: Date;
        staticData?: IDataObject;
        updatedAt?: Date;
        versionId?: string;
    }
